# Catholic translation glossary — target English

The translating agent MUST respect the following equivalences strictly, even
when stylistic intuition suggests rewording.

## Latin / Greek → English

### Grace and justification

- `gratia habitualis` → habitual grace (also: sanctifying grace when context demands)
- `gratia sanctificans` → sanctifying grace
- `gratia actualis` → actual grace
- `gratia praeveniens` → prevenient grace
- `gratia cooperans` → cooperating grace
- `gratia sufficiens` → sufficient grace
- `gratia efficax` → efficacious grace
- `iustificatio` → justification
- `meritum de condigno / de congruo` → condign merit / congruous merit

### Dogmatics and condemnations

- `anathema sit` → let him be anathema (NEVER soften to "let him be rejected")
- `si quis … dixerit … anathema sit` → if anyone shall say … let him be anathema
- `ex cathedra` → ex cathedra (INVARIANT)
- `infallibilitas` → infallibility
- `de fide definita` → de fide (defined)
- `depositum fidei` → deposit of faith
- `sensus fidei` → sensus fidei (INVARIANT)

### Sacraments

- `transubstantiatio` → transubstantiation (NEVER merely "change" or "conversion")
- `consubstantialis Patri` → consubstantial with the Father
- `praesentia realis` → Real Presence
- `ex opere operato / operantis` → ex opere operato / operantis (INVARIANT)
- `character indelebilis` → indelible character
- `baptismus fluminis / flaminis / sanguinis` → baptism of water / of desire / of blood
- `matrimonium ratum et consummatum` → ratum et consummatum marriage

### Ecclesiology

- `subsistit in` → subsists in (NEVER "is identical to")
- `extra Ecclesiam nulla salus` → outside the Church there is no salvation
- `Ecclesia militans / patiens / triumphans` → Church militant / suffering / triumphant
- `corpus Christi mysticum` → Mystical Body of Christ
- `communio sanctorum` → communion of saints
- `Filioque` → Filioque (INVARIANT)
- `Romanus Pontifex` → Roman Pontiff
- `Sancta Sedes` → Holy See
- `collegium episcoporum` → college of bishops

### Christology / Mariology

- `Theotokos` → Mother of God (or Theotokos where Greek context is load-bearing)
- `unio hypostatica` → hypostatic union
- `Immaculata Conceptio` → Immaculate Conception
- `Assumptio corporalis` → bodily Assumption
- `semper Virgo` → ever-Virgin
- `virginitas in partu` → virginity in partu (INVARIANT)

### Scripture and Tradition

- `Sacra Scriptura` → Sacred Scripture
- `Traditio` → Tradition (capitalized when referring to THE Tradition)
- `sola scriptura` → sola scriptura (INVARIANT)

### Liturgy

- `Novus Ordo Missae` → Novus Ordo Missae (INVARIANT)
- `vetus Ordo` / `usus antiquior` → traditional Latin Mass / usus antiquior
- `lex orandi, lex credendi` → lex orandi, lex credendi (INVARIANT)
- `pro multis` → pro multis (INVARIANT)

### Morals

- `actus humanus` → human act
- `peccatum originale` → original sin
- `concupiscentia` → concupiscence
- `virtus infusa / acquisita` → infused virtue / acquired virtue
- `lex naturalis` → natural law
- `ius naturae` → natural right

### Salvation and eschatology

- `visio beatifica` → beatific vision
- `novissima` → Last Things
- `purgatorium` → purgatory
- `limbus puerorum / patrum` → limbo of the children / of the fathers

## Italian → English

- `Magistero` → Magisterium
- `Santo Padre` → Holy Father
- `dottrina cristiana` → Christian doctrine
- `Catechismo` → Catechism

## French → English (when source is French)

- `Magistère` → Magisterium
- `Saint-Père` → Holy Father
- `concile œcuménique` → ecumenical council
- `Siège apostolique` → Apostolic See

## Transverse formatting rules

- Canon numbering: keep Roman numerals when the source uses them
  ("Canon III", not "Canon 3").
- Scriptural references: keep the standard Latin abbreviations
  (`Mt`, `Lc`, `Jn`, `Rom`, `1 Cor`…) — do not translate them.
- Biblical quotations inside running text should follow the Douay-Rheims
  tradition (not NRSV / NABRE which paraphrase).
- "Vatican II" stays "Vatican II" (not "the Second Vatican Council" mid-text).
- When a Latin phrase is used in italics in the source, keep it in Latin
  italics in English, with a parenthetical English gloss on first
  occurrence only, then bare Latin thereafter.
