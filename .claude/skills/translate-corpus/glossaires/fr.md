# Glossaire de traduction catholique vers le français

L'agent traducteur DOIT respecter strictement les équivalences ci-dessous,
même quand son intuition stylistique le pousserait à reformuler.

## Latin / grec → français

### Grâce et justification

- `gratia habitualis` → grâce habituelle
- `gratia sanctificans` → grâce sanctifiante
- `gratia actualis` → grâce actuelle
- `gratia praeveniens` → grâce prévenante
- `gratia cooperans` → grâce coopérante
- `gratia sufficiens` → grâce suffisante
- `gratia efficax` → grâce efficace
- `iustificatio` → justification
- `meritum de condigno` → mérite de condignité (invariant en théologie)
- `meritum de congruo` → mérite de congruité

### Dogmatique et condamnations

- `anathema sit` → qu'il soit anathème
- `si quis … dixerit … anathema sit` → si quelqu'un dit … qu'il soit anathème
- `ex cathedra` → ex cathedra (INVARIANT)
- `infallibilitas` → infaillibilité
- `definitio fidei` → définition de foi
- `de fide definita` → de foi définie
- `doctrina catholica` → doctrine catholique
- `sensus fidei` → sensus fidei (INVARIANT)
- `depositum fidei` → dépôt de la foi

### Sacrements

- `transubstantiatio` → transsubstantiation (JAMAIS « conversion » seul)
- `consubstantialis Patri` → consubstantiel au Père
- `species panis et vini` → espèces du pain et du vin
- `praesentia realis` → présence réelle
- `ex opere operato` → ex opere operato (INVARIANT)
- `ex opere operantis` → ex opere operantis (INVARIANT)
- `character indelebilis` → caractère indélébile
- `baptismus fluminis` → baptême d'eau
- `baptismus flaminis` → baptême de désir
- `baptismus sanguinis` → baptême de sang
- `matrimonium ratum et consummatum` → mariage rat et consommé
- `matrimonium in fieri / in facto esse` → mariage in fieri / in facto esse

### Ecclésiologie

- `subsistit in` → subsiste dans (JAMAIS « est identique à »)
- `extra Ecclesiam nulla salus` → hors de l'Église, pas de salut
- `Ecclesia militans / patiens / triumphans` → Église militante / souffrante / triomphante
- `corpus Christi mysticum` → Corps mystique du Christ
- `communio sanctorum` → communion des saints
- `Filioque` → Filioque (INVARIANT)
- `Romanus Pontifex` → le Pontife romain
- `Sancta Sedes` → le Saint-Siège
- `collegium episcoporum` → collège des évêques

### Christologie / mariologie

- `Theotokos` → Mère de Dieu (ou Theotokos si le contexte grec est important)
- `hypostasis` → hypostase
- `unio hypostatica` → union hypostatique
- `Immaculata Conceptio` → Immaculée Conception
- `Assumptio corporalis` → Assomption corporelle
- `semper Virgo` → toujours Vierge
- `virginitas in partu` → virginité in partu (INVARIANT)

### Écriture et tradition

- `Sacra Scriptura` → Sainte Écriture
- `Traditio` → Tradition (avec majuscule quand c'est LA Tradition)
- `sola scriptura` → sola scriptura (INVARIANT, précisément parce que c'est une thèse protestante)
- `inspiratio` → inspiration
- `inerrantia` → inerrance

### Liturgie

- `Novus Ordo Missae` → Novus Ordo Missae (INVARIANT)
- `vetus Ordo` / `usus antiquior` → forme tridentine / usage antique (laisser le latin aussi)
- `lex orandi, lex credendi` → lex orandi, lex credendi (INVARIANT)
- `pro multis` → pro multis (INVARIANT)
- `Canon Missae` → Canon de la messe
- `oblatio munda` → oblation pure

### Morale

- `bonum honestum / utile / delectabile` → bien honnête / utile / délectable
- `actus humanus` → acte humain
- `peccatum originale` → péché originel
- `concupiscentia` → concupiscence
- `virtus infusa / acquisita` → vertu infuse / acquise
- `lex naturalis / naturae` → loi naturelle
- `ius naturae` → droit naturel

### Salut et eschatologie

- `oeconomia salutis` → économie du salut
- `visio beatifica` → vision béatifique
- `novissima` → fins dernières
- `purgatorium` → purgatoire
- `limbus puerorum / patrum` → limbes des enfants / des pères

## Italien → français

- `Magistero` → Magistère
- `Santo Padre` → Saint-Père
- `dottrina cristiana` → doctrine chrétienne
- `Catechismo` → Catéchisme
- `Sacrosanto Concilio` → Saint Concile

## Anglais → français (quand la source est en anglais)

- `magisterium` → Magistère (majuscule quand c'est l'institution)
- `Holy See` → Saint-Siège
- `Roman Pontiff` → Pontife romain
- `see of Rome` → siège de Rome
- `apostolic see` → Siège apostolique
- `ecumenical council` → concile œcuménique

## Règles de formatage transversales

- Les numéros de canon restent en chiffres romains si c'est l'usage de la
  source : « Canon III » pas « Canon 3 ».
- Les références scripturaires gardent l'abréviation latine usuelle :
  `Mt`, `Lc`, `Jn`, `Rom`, `1 Cor`, etc. — pas de traduction.
- Les citations bibliques à l'intérieur du texte doivent être traduites
  selon la Vulgate Sixto-Clémentine ou la Bible de Crampon / Fillion
  (jamais la Bible de Jérusalem ou la TOB qui modernisent).
- Les titres en latin dans les in-text italics restent en latin italisé,
  avec traduction française entre parenthèses en première occurrence
  seulement. Ensuite, garder le latin seul.
- « Vatican II » sans traduction (invariant francophone). Pas « le deuxième
  concile du Vatican » en cours de texte.
